# ZhengFang-Auto-Evaluation (正方教务全自动避测评教脚本)

🚀 **一个基于 JavaScript 的正方系统全自动评教辅助脚本，支持多科目循环、自动避测及弹窗自动确认。**

## ✨ 特性
- **全自动循环**：一键处理“未评”和“未评完”的所有课程，无需手动切换。
- **高仿真避测**：模拟完整指针事件流（Pointer Events），规避 `isTrusted` 自动化检测。
- **智能穿透**：自动识别并进入评价所需的 `iframe` 容器。
- **极速稳定**：优化了操作间隔，并在提交环节增加了系统容错处理，解决 jQuery 插件报错问题。
- **自定义评语**：默认填入“无”，可自行修改。

## 🛠️ 使用方法
1. 登录教务管理系统，进入 **“学生评教”** 或 **“教学评价”** 列表页面。
2. 按下 `F12` 键打开开发者工具，点击 **Console（控制台）** 选项卡。
3. 将 `src/script.js` 中的代码全部复制并粘贴到控制台中。
4. 按下 `Enter` 回车键，脚本将自动开始运行。
5. **注意**：脚本运行期间请勿关闭页面，直到弹出“全部评价完成”的提示。
6. 如果需要停止脚本，则使用 `F5` 来刷新页面中止程序。
## ⚙️ 常见问题 (FAQ)
- **Q: 运行过程中控制台报错 `Uncaught TypeError`？**
  - A: 这是系统自带 jQuery 插件在关闭弹窗时的内部冲突，脚本已做容错处理，不影响评价结果。
